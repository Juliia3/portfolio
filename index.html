React project.

This is a landing page made with React for an Interior Design brand. It has all the main features that modern websites have like routing, responsive design, image sliders, and more. Besides React, this website also uses the Swiper library and its elements are styled with SCSS

Design

React project.

This is a project made with React about Home Interior Design brand. It contains React Router to help our user with navigation to other pages, Swiper library that lets user to slide images and see more images and cards. Components and elements are styled with SCSS. There is a basket with products that you can add there pressing a special button, that was made with React Context.

Boutique

Vue project

This is an internet store website made with Vue. It contains Vue Router to make navigation to other pages, API, that gives us products, their description etc. It has a filter to select what kind of goods you want to look for. Components and elements are styled with SCSS. Also in this project I used Vuex which let me manage my elements' states to show more information about each product by pressing "read more". You can tap on every card and get more information about each one.
